# 隐微分求解 | Implicit differentiation

## 方法

将方程转化为二元函数，再通过公式
$$$隐微分求解公式
\frac{dy}{dx} = -\cfrac {\frac{\partial F}{\partial x}}{\frac{\partial F}{\partial y}} = - \frac{F_x}{F_y}
$$$
进行求解。

## 例题

Find $$y'$$ if $$x^3 + y^3 = 6xy$$

设 $$F(x, y) = x^3 + y^3 - 6xy = 0$$
由上述公式 $$\frac{dy}{dx} = - \frac{F_x}{F_y}$$ 即可求解。
